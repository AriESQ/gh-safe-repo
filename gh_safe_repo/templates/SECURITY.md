# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

**Do not open a public GitHub issue for security vulnerabilities.**

Instead, please email: {security_email}

Include:
- A description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested fixes (optional)

You can expect an acknowledgement within 48 hours and a substantive response within 5 business days.

## Security Updates

Security fixes are released as soon as possible after a vulnerability is confirmed. Subscribe to repository notifications to be informed of security releases.
